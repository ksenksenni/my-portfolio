Ксения Николаева

HTML-верстальщик

Привет, меня зовут Ксюша, я живу в Санкт-Петербурге и занимаюсь HTML-версткой сайтов. Я создаю полностью адаптивные кросбраузерные сайты с валидной версткой. До всего этого, я занималась интернет-маркетингом, в том числе анализом пользовательского поведения, поэтому при работе над сайтом, я большое внимание уделяю UX.

Технологии, которыми я пользуюсь

HTML5

Я использую семантическую верстку, чтобы разметка была логичной, простой и понятной, что позволяет экономить время и нервы.

BEM

Использую методологию BEM при именовании классов для упрощения работы с кодом, что позволяет, опять-таки, экономить время и нервы.

Bootstrap

Я использую популярный фреймворк, чтобы сэкономить время при работе над проектом и упростить его дальнейшее развитие.

CSS3

Создаю красоту по макетам: различные сетки (flex, grid), резиновая адаптивная верстка, анимация, svg-изображения.

SASS/SCSS

С помощью препроцессоров ускоряю разработку сайта, улучшаю логику стилей и упрощаю синтаксис.

Figma/ Scetch/ Photoshop

Работаю с макетами в самых популярных программах (нарезка изображений, экспорт SVG, создание спрайтов).

Javascript

С помощью JS создаю динамические элементы: формы, меню, калькуляторы, всплывающие окна и т.п.

Gulp

Использую для сборки ассетов, компиляции и минификации кода для того, чтобы работа над проектом была более простая и быстрая.

GIT

Система версий — полезный инструмент, всегда можно быстро откатить назад при необходимости, а при работе в команде вообще незаменим.

Лендинг для food-приложения

Адаптивный лендинг на флексах с использованием препроцессора SCSS и сборщика Gulp с семантической разметкой и именованием классов по методологии BEM.

Страница для интернет-магазина

Адаптивная страница с карточками товаров, с настроенной на JS сортировкой их по различным параметрам, блокировкой формы при вводе некорректного email, кнопкой плавной прокрутки наверх и всплывающими окнами информации. Реализация бургер-меню на чистом CSS.

Форма с проверкой длины пароля

Адаптивный блок с формой: отправка данных на сервер, всплывающие окна и блокировка отправки формы при введении пароля короче 4 символовов.
